Welcome to your first steps into the Arduino world! In this guide, we’ll cover everything from installing the IDE to running your first program.

Installing the Arduino IDE

Download the IDE from the official Arduino website. Available for Windows, macOS, and Linux.

// Example Blink Sketch
void setup() {
pinMode(LED_BUILTIN, OUTPUT);
}
void loop() {
digitalWrite(LED_BUILTIN, HIGH);
delay(1000);
digitalWrite(LED_BUILTIN, LOW);
delay(1000);
}

Serial Monitor Basics

The Serial Monitor is your debugging friend. Use Serial.begin(9600); inside setup() to start it.

Safety Checklist

  • Always double-check wiring before plugging into USB.
  • Never draw too much current from I/O pins.
  • Use resistors with LEDs to prevent damage.
